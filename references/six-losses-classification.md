# 六大损失分类与归因方向

> 本文件为 Agent 解读六大损失、定位改善重点的知识依据。

## 一、六大损失与三大指标映射

| 损失类型 | 具体表现 | 归类 | 对应 OEE 指标 |
|----------|----------|------|------|
| 设备故障损失 | 宕机、重大故障 | 可用性损失 | 时间开动率 |
| 换模调整损失 | 换型调试、试产 | 可用性损失 | 时间开动率 |
| 空转暂停损失 | 待料、断料、堵料 | 性能损失 | 性能开动率 |
| 减速损失 | 设备老化降速、参数未优化 | 性能损失 | 性能开动率 |
| 启动损失 | 开机预热、首件调试 | 性能损失 | 性能开动率 |
| 质量损失 | 不良品、返工品 | 质量损失 | 合格品率 |

## 二、各类损失的典型归因方向

- **设备故障**：MTBF 偏低、点检不到位、备件管理缺失 → 建 MTBF/MTTR 追踪、周保全点检闭环
- **换模调整**：内部/外部作业未分离 → 导入快速换模(SMED)，目标换型时间减半
- **空转暂停**：物料齐套性差 → 物料预警机制、拉动补料
- **减速损失**：设备老化、工艺参数保守 → 排查降速根因、优化参数
- **启动损失**：首件确认慢、预热流程长 → 标准化开机检查表
- **质量损失**：工艺控制点漂移、首件未确认 → 首件确认机制、SPC 控制

## 三、改善优先级判定逻辑

当六大损失数据完整时，优先改善**占比最大**的损失项；当仅三大指标可得时，优先改善**与 100% 差距最大**的那个维度（时间/性能/质量）。

> 注：改善措施为通用建议，具体实施需结合现场条件，且措施预期效果不应作为承诺值呈现。
